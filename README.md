## Features

- Create a trip
- Invite guests
- Create activities for a trip
- Manage guests
- Manage activities

## Requirements

- [Java]([https://nodejs.org/en/](https://docs.oracle.com/en/java/)) >= 21
- [Maven]([https://pnpm.js.org/](https://maven.apache.org/guides/index.html)) >= 8.10.2
